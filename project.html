GPTGreen

Description:

    GPTGreen is a project that aims to quantify and visualize the energy consumption of GPT models. The project tracks the number of tokens that GPT produces during output and counts the input tokens as well. Using this data, it calculates the energy required for each token in Joules, offering insight into the environmental impact of AI models. The energy consumption is then displayed live within the GPT interface, allowing users to see the real-time energy cost of generating text. This initiative raises awareness of the energy footprint of AI systems and encourages more sustainable practices in AI model usage.

Image Compression

Description:

   This project focuses on an image compression technique where the image is initially converted from RGB (Red, Green, Blue) color space to a more efficient representation using CV (Color Vectorization). The image is then transformed into the Discrete Cosine Transform (DCT) domain to further compress the data. The project also includes a reversal process where the compressed image is transformed back, going through the DCT to CV, and then back to the original RGB color space. This method reduces the file size while maintaining image quality, offering an efficient approach for storing and transmitting images.

ASL Recognition

Description:

   This project is an American Sign Language (ASL) recognition system that leverages computer vision and machine learning to classify both static and dynamic ASL characters. It utilizes Mediapipe to extract hand landmarks from video frames and Keras to train a Multi-Layer Perceptron (MLP) network for classification. The model can recognize static gestures as well as dynamic signs like 'J' and 'Z,' enhancing accessibility for ASL users. This project demonstrates the potential of AI-driven gesture recognition for real-time communication and assistive technologies.
